### Install

1. Install [Ruby 2.3.1 or newer](https://www.ruby-lang.org/)
2. `gem install bundle`
2. Clone repo and `cd`
    + Run `bundle install`
    + Run `bundle exec middleman server`

You can now see the docs at http://localhost:4567
